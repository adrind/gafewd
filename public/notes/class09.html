JS Review

HTML + CSS + JS = Websites!

• HTML:

⇒ Used to add and structure content like text and images on a page.

• CSS:

⇒ Used to style and position the page content.

• JavaScript:

⇒ Used to respond and change the content based on user interaction.

JavaScript: History

• JavaScript was created in 10 days in 1995.

• Today it is the language we use to change content on the page in web browsers.

• It's gone on to become one of the most popular languages in general: also used to develop server applications and large complex frontend applications with JavaScript frameworks.

JavaScript: Uses

Most uses of JavaScript in the browser fall into four categories:

1) Responding to user actions (events)

2) Changing HTML Content after the page has loaded

3) Changing CSS Styles

4) Sending/retrieving data to/from the server

JavaScript: Uses

JavaScript consists of sequences of commands.

Examples:

• If a user clicks this button, change the background color of the page.

• If a user enters text in an input box, check to see if it's a valid email format.

JavaScript: Types

JavaScript lets us store pieces of data to use throughout our program.

Examples:

• Track of how many times a user clicked a button

• An email address and password a user entered into a text box

JavaScript: Types

• Data types:

⇒ How we classify data (Strings, Numbers, Booleans, etc.)

⇒ Determines how we interact with the data

• Variables:

⇒ Placeholder used to store a piece of data


	var x = 2;

⇒ How we manage state (store memory) in a program

⇒ Assignment only occurs left to right!

JS Reivew: HTML Queries

document.querySelector()

⇒ Queries HTML for the first element with the specified CSS selector


	document.querySelector('#logo')

⇒ The above will look through a HTML document and return the first element with id="logo"

⇒ Once an element is returned, additional JavaScript methods can be "called" on it

CodePen Notes

JS Reivew: HTML Queries

• Querying Examples:


	// will return the first element with an ID of "logo"
	document.querySelector('#logo');

	// will return the first <h1> element
	document.querySelector('h1');

	// will return all <p> elements
	document.querySelectorAll('p');

	// will return all elements with a class of "feature"
	document.querySelectorAll('.feature');

• Note: document.querySelectorAll returns a list of elements

Browser Events: Exercise

• We can use JavaScript to track and respond to events that a user takes on a website.

• Example: the user clicks on a button.

• Take a couple minutes with a partner to list as many interactions you can have with a webpage that you can think of.

Browser Events: Overview

• Events that users perform in the browser:

⇒ click

⇒ mouse over / mouse out

⇒ key down / key down

⇒ form submit

• We can use JavaScript to track and respond to events

Browser Events: Applying Events

1) Use document.querySelector() to select the element you want to apply the event to:


	document.querySelector('#click-me')

2) Apply the event name (ex: .onclick) to the above query:


	document.querySelector('#click-me').onclick

3) Set the above equal to the function that will run when the event is triggered


	document.querySelector('#click-me').onclick = myFunc

Browser Events: Mouse Over/Out

• Use .onmouseover and .onmouseout to achieve a hover effect:


	var button = document.querySelector('#my-btn');

	button.onmouseover = overFunction;
	button.onmouseout = outFunction;

⇒ .onmouseover is triggered when a user moves the cursor over the desired HTML element

⇒ .onmouseout is triggered when a user's cursor leaves said element

innerHTML Property

innerHTML Property

• .innerHTML is used to read and write HTML

• Assuming the following HTML:


	<div id="result">Hello, World!</div>

⇒ To read:


	document.querySelector('#result').innerHTML;
	=> 'Hello, World!'

⇒ To write:


	document.querySelector('#result').innerHTML = 'Cats';

parseInt() Function

parseInt() Function

• .innerHTML returns a string value!

• parseInt() converts the string data type to a number data type


	var x = "2"

	x + 2
	// "22"

	x = parseInt(x)
	x + 2
	// 4
