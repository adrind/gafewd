CSS Intro

CSS: History

• CSS => Cascading Style Sheets

• First proposed in 1994 by Håkon Lie ("How Con Lee") - a colleague of Tim Berners-Lee

• Stylesheet language used to describe HTML markup

• Allows separation of document content (HTML), from document styling (CSS)

⇒ "Separation of concerns"

• Start a new CodePen

CSS: Syntax

• This CSS rule turns the text color of every paragraph, <p>, black

CSS: Syntax (ctn)

• One selector can have multiple declarations

• Common for each declaration to have its own line

CSS Color Property

CSS Color Property

• Text color can be set with color:


	h1 {
	  color: red;
	}

• Color values can be specified using:

⇒ color keyword: red

⇒ rgb: rgb(255, 0, 0)

⇒ hex code: #FF0000

CSS Color: Keyword Value

	span {
	  color: red;
	}

• Simplest; least-descriptive

• red, green, blue, etc.

⇒ deeppink

⇒ chartreuse

• More colors...

CSS Color: RGB Value

	span {
	  color: rgb(255, 0, 0);
	}

• Bit more complicated; bit more powerful

• First value is red; second is green; third is blue

• Each value in a range 0 - 255

• Values closer together are gray or muted:

rgb(200,200,200) / rgb(155,155,155) / rgb(075,075,075)

CSS Color: Hexadecimal Value

	p {
	  color: #463DB3;
	}

• Very robust...but there's some maths happening

• How it works: "The Code Side of Color"

• Adobe Color: https://color.adobe.com

Applying Color: Text vs. Background

• color ⇒ Used to apply color to text


	p {
	  color: rgb(255, 0, 0);
	}

• background ⇒ Used to apply a background color (or image) to elements:


	p {
	  background: rgb(0, 255, 0);
	}

Let's review!

CSS Font: font-weight

• font-weight

⇒ Sets the "thickness" of the font

⇒ You can use it on p or div or any element that has text

⇒ Can take keyword or numerical values:

• normal and bold eg: font-weight: bold;

• or values 100 - 900 eg: font-weight: 600;

CSS Font: font-style

• font-style

⇒ Sets a "slant" to the font

⇒ You can use it on p or div or any element that has text

⇒ Can take keyword values:

⇒ normal and italic

• Eg: font-style: italic;

CSS Style: border

• border

⇒ Adds a border to an element. You can specify how thick, the type, and the color in one property.

⇒ You can use it on any element!

⇒ Requires three values or you can break it up using seperate property names.

• Eg: 

div {
  border: 2px solid red;
}

• Eg (same result!): 

div {
  border-width: 2px;
  border-color: red;
  border-type: solid;
}

CSS Font: font-size

• font-size

⇒ Sets the size of the font

⇒ You can use it on p or div or any element that has text

⇒ Can take pixel values, percentages, or "ems":

• Eg: font-size: 28px;

• Eg: font-size: 200%;

• Eg: font-size: 1.5em;

CSS: list-style

• list-style

⇒ Changes how the bullet points of a list are styled.

⇒ You can only use it on li or ul/ol

⇒ Can use list-style as a shorthand for three list style properties: list-style-type, list-style-position, list-style-image

• Eg: 

ul {
  list-style: square inside url(https://image.flaticon.com/icons/png/128/39/39783.png);
}

• Eg (same result!): 

ul {
  list-style-type: square;
  list-style-position: red;
  list-style-image: url(https://image.flaticon.com/icons/png/128/39/39783.png);
}

CSS Font: font-family

• font-family

⇒ Applies a particular font to the element

⇒ You can use it on all elements that have text inside them

⇒ Should always end with: serif, sans-serif, or monospace

Pick the most specific font and add "fallbacks" that you know are more common and are installed on most computers.

You can always include files of special fonts to use fonts that are not already installed in the browser.


	p {
	  font-family: 'Comic Sans MS', helvetica, serif, sans-serif;
	}
CSS: padding

• padding

⇒ Adds space between the content of an element and its border.

⇒ You can only use it on any element

⇒ Can use padding as a shorthand for four padding properties: padding-left, padding-right, padding-top, padding-bottom

• Eg: 

p {
  padding: 20px;
}

• Eg (same result!): 

p {
  padding-top: 20px;
  padding-bottom: 20px;
  padding-left: 20px;
  padding-right: 20px;
}

CSS Font: text-align

• text-align

⇒ how text is oriented within the containing block

⇒ can be used on any element with text

⇒ only accepts a small list of keyword values like left, center, or right


	p {
		text-align: left;
	}

• text-decoration

⇒ controls extra-textual visual emphasis (underline)


	p {
	  text-decoration: underline;
	}

Working with CSS

Working with CSS

• Three ways to apply CSS to HTML:

1) Inline CSS

2) Embed in HTML using a <style> tag

3) Link to an external file using the <link> tag

Inline CSS

	<p style="color: blue; font-size: 14px;">
	  Chocolate chip ice cream is better than Chocolate ice cream.
	</p>

	<p style="color: blue; text-decoration: underline;">
	  But Cookies N'Cream is better than both of them.
	</p>

• Not a good practice

• Does not separate our concerns (HTML/CSS)

• Makes debugging difficult

• ONLY USE IF WRITING HTML EMAILS

• You will make enemies

Do NOT Use Inline CSS!

<head> embed

	<!DOCTYPE html>
	<html>
	<head>
	  <title>Super Awesome Website</title>
	  <style type="text/css">
	    h1 {color: red;}
	    p {color: blue; font-size: 14px}
	  </style>
	</head>
	<body>
	  ...

• Better than inline styles, as styles share same location

• Bloats HTML pages

• Can make debugging large applications difficult

External CSS

	<!DOCTYPE html>
	<html>
	<head>
	  <title>Super Awesome Website</title>
 	  <link rel="stylesheet" type="text/css" href="style.css">
	</head>
	<body>
	  ...

• Like <title>, placed within <head>

• Truly separates HTML from CSS

• Easiest to debug; just go to your CSS file!

External CSS

     <link rel="stylesheet" type="text/css" href="style.css">

• rel

⇒ Specifies the relationship between the current document and the linked document

• type

⇒ Specifies the media type of the linked document

• href

⇒ Specifies the location of the linked document
