Google Maps API

What is an API?

• API ⇒ Application Programming Interface

• Set of rules/instructions for interacting with a piece of software

• Middleman; allows different pieces of software to "talk" to each other

API vs Plugin
API

• A way to get/send data to/from another piece of software

• Up to the developer to style/add the data to their website

• Often requires permissions to use

Plugin

• Gives you a file or code to add to your page

• Usually comes with some configured styles/functionality so a developer doesn't have to do much work

• Doesn't necessarily talk to another piece of software -- stand alone code snippet

Facebook plugins vs Facebook API

Arrays

Arrays

• Arrays are a list-like data structure of comma-separated values

• Useful for storing a sequence of related data

• Declare arrays using square brackets: [ ]


	var friends = ['Moe', 'Larry', 'Curly'];

• Element - an item inside the array

• Index - the numerical location of each element in the array (starts at position 0 - aka "0-indexed")

Arrays

• JS provides multiple methods for array creation:

⇒ The new keyword


	var drinks = new Array('beer', 'coffee', 'water');

⇒ Using the array literal


	var drinks = ['beer', 'coffee', 'water'];

• While similar, there are crucial differences

• Use the array literal for now

Arrays

• Use [index] to retrieve values from an array

• Remember that arrays are 0-indexed


	var drinks = ['beer', 'coffee', 'water'];

	drinks[0]
	// 'beer'

	drinks [0+1]
	// 'coffee'

	var x = drinks.length - 1
	drinks[x]
	// 'water'

	drinks[4]
	// undefined

Array Iteration

Array Iteration

• Iteration is the process of running the same function to every element in an array:


  var neighborhoods = ['the mission', 'nob hill', 'inner sunset', 'north beach', 'bayview'];

  neighborhoods.forEach(logNeighborhood);

  function logNeighborhood (neighborhood) {
    console.log(neighborhood);
  }

  // 'the mission'
  // 'nob hill'
  // 'inner sunset'
  // 'north beach'
  // 'bayview'
Array Iteration

• JS arrays have several iteration methods:

.forEach()

.every()

.some()

.filter()

.map()

• Most require a function to be supplied as an argument; this function will be applied to each item in the array, individually

Objects

• JS has another complex data type: an Object

• Object is a way to store related bits of data in one variable

⇒ Not using an object

var dogName = 'Rocky';
var dogBreed = 'Corgi mix';
var dogColor = 'Black';

⇒ Using an object

var dog = {
  'name': 'Rocky',
  'breed': 'Corgi mix',
  'color': 'black'
};

console.log(dog.name); //This will say "Rocky"
console.log(dog['name']); //Also "Rocky"
Objects

• We can create Objects that have Arrays in them:

var dog = {
  'name': 'Rocky',
  'breed': 'Corgi mix',
  'favoriteThings': ['food', 'attention', 'purple ball']
};

• Or create Arrays of Objects:

var dogs = [
  {'name': 'Rocky', 'breed': 'Corgi mix'}, 
  {'name': 'Romeo', 'breed': 'German Shepherd'}
];
Arrays vs Objects

• Arrays are lists of data

var food = ['orange', 'banana', 'apple'];

var firstFood = food[0]; //How to get a value from the list

food.push('pineapple'); //How to add a value to the list

food[1] = 'strawberry'; //How to change a value in a list

Now the list is ['orange', 'strawberry', 'apple', 'pineapple']

Arrays vs Objects

• Objects store key:value pairs of information

var food = {'name': 'orange', 'type': 'fruit', 'price': '1 dollar'};

var foodName = food.name; //How to get a value from the object

food['flavor'] = 'sweet'; //How to add a new key:value pair to the object

food['price'] = '2 dollars'; //How to change a value in the object

Now the object is {'name': 'orange', 'type': 'fruit', 'price': '2 dollars', 'flavor': 'sweet'}
