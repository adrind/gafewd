$(this)

• $(this) allows us to target specific elements that invoked (or called) a function

• Refactor previous CodePen using $(this):


	function changeColor() {
	  $(this).css('color', 'red');
	}

	$('span').click(changeColor);

• On the click event, only the span that invoked the function will turn red

$(this)

• In JS, functions are called (invoked) by an object

• Whatever object called the function can be accessed inside the function using this

• this is said to be the "context" of the function (refers to whatever object invoked/called it)

• We can "wrap" this in jQuery: $(this)

• Remember, DOM nodes (HTML elements) are objects

If/Else Flow:

⇒ AKA conditional or selection

⇒ Used to alter program flow when a decision must be made between two or more actions

⇒ This choice is usually made based on variables within the function/program

Conditional Statements

Conditional Flows JavaScript Syntax:


	var x = 10;

	if (x > 5) {
	  alert('x is larger than 5!');
	} else {
	  alert('x is smaller than 5!');
	}

• Denoted by the if and else keywords

• Begins with a condition: (x > 5)

• If the condition is true, the statment(s) in the "if block" will run

• If the condition is false, the statment(s) in the "else block" will run

Manipulating CSS Classes

Manipulating CSS Classes

.addClass('class-name')

⇒ Adds classes to elements


	$('p').addClass('bold-text')

	// would yield: <p class="bold-text"></p>

.removeClass('class-name')

⇒ Removes classes from elements


	$('p').removeClass('bold-text')

⇒ Adds class to elements if it does not exist; otherwise removes
