Toggling Element Visibility

Toggling Element Visibility

• Hiding/showing HTML is a very common effect

• Allow developers to render more content w/out scroll

• Useful for menus/tooltips/modals (overlays)

• Hidden elements are not destroyed; they are just not visible

• Ex: Bootstrap modal

jQuery's .hide() / .show()

.hide()

⇒ Hides elements by setting their inline style to display: none;


	$('p').hide();

.show()

⇒ Reveals elements by setting their inline style to display: block

⇒ If the element originally had display: inline, then display: inline will be used


	$('p').show();
jQuery's .hide() / .show()

• Change the duration by passing an argument:

⇒ String:


	$('p').hide('slow')

	$('p').show('fast')

⇒ Number:


	$('p').hide(700)

	$('p').show(300)
jQuery's .hide() / .show()

• jQuery provides alternatives to .hide() and .show():

.fadeIn() and .fadeOut()

.slideDown() and .slideUp()

.slideToggle()

• Each takes an optional duration argument:


	$('p').fadeIn('fast')

	$('p').slideUp(900)

Manipulating Element Attributes

Manipulating Element Attributes

.attr('attribute-name')

⇒ Gets the value of the matched attribute name


	<img src="cute-puppy.jpg">

	$('img').attr('src')
	// returns 'cute-puppy.jpg'

.attr('attribute-name', 'attribute-value')

⇒ Changes the value of 'attribute-name' to 'attribute value'


	$('img').attr('src', 'happy-puppy.jpg')

	<img src="happy-puppy.jpg">
Manipulating Element Attributes

• Assuming the following HTML:


	<a href="http://google.com">Google
	<img id="logo" src="smiley.png">

• Get attribute .attr('attribute-name'):


	$('a').attr('href')

	// returns 'http://google.com'



	$('#logo').attr('id')

	// returns 'logo'
Manipulating Element Attributes

• Assuming the following HTML:


	<img id="logo" src="smiley.png">

• Set attribute .attr('attribute-name', 'attribute-value'):


	$('#logo').attr('src', 'frowney.png')

• Turns the above HTML into:


	<img id="logo" src="frowney.png">

$(this)

• $(this) allows us to target specific elements that invoked (or called) a function

• Refactor previous CodePen using $(this):


	$('span').click(function () {
	  $(this).css('color', 'red')
	})

• On the click event, only the span that invoked the function will turn red

$(this)

• In JS, functions are called (invoked) by an object

• Whatever object called the function can be accessed inside the function using this

• this is said to be the "context" of the function (refers to whatever object invoked/called it)

• We can "wrap" this in jQuery: $(this)

• Remember, DOM nodes (HTML elements) are objects

Document Traversal

Document Traversal

• Given a certain element, how do I access elements around it?

• jQuery makes it very easy to move around the DOM

• "Walking the DOM"

• Details: http://api.jquery.com/category/traversing/

• What do you think the following will do?


	$('#wrapper').children()

• Returns the elements nested inside of #wrapper

Document Traversal: It's all in the family

.children()

⇒ Get an element's child elements

.parent()

⇒ Get an element's parent element

.siblings()

⇒ Get an element's sibling elements

Document Traversal: Siblings

.next()

⇒ Get the immediate following sibling element

.prev()

⇒ Get the immediate preceding sibling element

Document Traversal: Find
