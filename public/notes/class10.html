Ingredients

• In JavaScript this is our "raw" data

• This raw data falls into the data types we learned about:

⇒ Numbers (6, 15, -2, 3.4)

⇒ Strings ("Hello!!", "I am a cat", "6")

⇒ Booleans (true, false)

Salad Bowls

• We use variables to keep track of our data as we change it

var saladBowl = "lettuce";

• As we go through our program we can use variables to store data 

var saladBowl = "lettuce";
saladBowl = "lettuce" + "tomato";

Salad Bowls (aka variables)

• You can assign the value of one variable to a different variable.

var smallBowl = "chopped tomato";
var largeBowl = "lettuce";
var saladBowl = smallBowl + largeBowl;

//value of saladBowl is now "chopped tomatolettuce"

Chopping

• Functions are used to transform the raw data into a new form we want.

function chopTomatoes() {
	var numberOfTomatoes = 2;
	numberOfTomatoes = numberOfTomatoes * 2;
	document.write("You now have " + numberOfTomatoes + " tomato pieces");
}

Chopping (aka Functions)

• Functions can take in data as input

function chopTomatoes(numberOfTomatoes) {
	numberOfTomatoes = numberOfTomatoes * 2;
	document.write("You now have " + numberOfTomatoes + " tomato pieces");
}


chopTomatoes(2); //You now have 4 tomato pieces
chopTomatoes(4); //You now have 16 tomato pieces
chopTomatoes(0); //You now have 0 tomato pieces

Chopping (aka Functions)

• Functions can also output data. We call this "returning" a value

function chopTomatoes(numberOfTomatoes) {
	numberOfTomatoes = numberOfTomatoes * 2;
	return numberOfTomatoes;
}


var tomatoPieces = chopTomatoes(2);
document.write("I'm done chopping! I have " + tomatoPieces + " tomato pieces");
document.write("I love my " + tomatoPieces + " tomato pieces");

Chopping (aka Functions)

• When we use a built-in function like document.querySelector("#myId") we are calling a function that takes an input ("#myId") and has an output (the element on the page)

• You can think of this like using some store bought ingredients

• Even though you can make your own salad dressing, it's sometimes easier to buy pre-made. You don't know how they made it but you know what the output is!

• This is what we're doing when we're using functions like document.write, parseInt

, and jQuery.
Browser Events

• When we assign a function to an event, we are giving it a task to do. 

function dressSalad() {

document.querySelector('#submitButton').onclick = chopTomatoes;

jQuery CSS Selectors

jQuery CSS Selectors: $('x')

• Use CSS selectors to query elements

• Similar to document.querySelector()

• Use the same selectors you write in CSS!

• The $('x') Function - Get all of the 'x'


	$('.nav-links') // select all elements with class "nav-links"
	$('h2') // select all <h2>'s
	$('p a') // select all <a>'s inside of all <p>'s
	$('p, a') // select all <p>'s and <a>'s

• The $('x') function takes ANY CSS selector

jQuery CSS Selectors: ID's and Classes

Fetch ID's using the # (hashtag):


	$('#someId')
	$('#someId a')

Fetch Classes using the . (period):


	$('.someClass')
	$('.someClass a')

Note: Using $('.someClass') returns ALL of the HTML elements with a class of someClass

jQuery CSS Selectors: Practice

$('div')

⇒ Select all HTML <div> elements

$('#someId')

⇒ Selects the first HTML with an ID of "someId"

$('.someClass')

⇒ Select all the HTML elements with class "someClass"

$('ul li .social-link')

⇒ Select elements with class social-link that are inside <li> elements which are inside <ul>

<input> and .val()

<input> and .val()

JavaScript:


	document.querySelector('#someId').value
	// returns the value inside of someId

jQuery:


	$('#someId').val()
	// returns the value inside of someId
<input> and .val()

• Getting values, use .val()


	$('#someId').val()
	// returns the value inside of someId

• Setting values, use .val('some text')


	$('#someId').val('some text')
	// sets the value of someId to 'some text'

.val() can be used on both <input> and <select> elements

Using jQuery

Using jQuery

Option 1: Download and store jQuery locally

1. Download jQuery from http://jquery.com/download

2. Save the .js file in your project's js directory

3. Create a <script> tag ABOVE any of your own JS files


	<script src="jquery.js"></script>
	<script src="project.js"></script>
Using jQuery

Option 2: Link to an external jQuery source (CDN)

1. Locate an externally hosted version of jQuery:

⇒ https://code.jquery.com/

2. Create a <script> tag ABOVE any of your own JS files

3. Point the src attribute to the jQuery CDN


	<script src="https://code.jquery.com/jquery-1.11.3.js"></script>
	<script src="project.js"></script>
Using jQuery: Script Placement

<script> tags should always be placed:

• At the BOTTOM of your <body>

• Right before the closing </body>; tag

Why:

• As your JS will reference DOM elements, these elements must be loaded first!

• By placing <script> tags at the BOTTOM, the browser will load these <script>'s LAST

CSS Manipulation

CSS Manipulation: .css()

JavaScript:


	document.querySelector('#someId').style.color = 'red'

jQuery:


	$('#someId').css('color', 'red')
CSS Manipulation: .css()

• Set Single Property:


	$('p').css('background-color', 'pink')

• Multiple Properties:


	$('#someId').css({ 'height': '100px', 'width': '100px' })

• Get/Read Property:


	$('#someId').css('height')
