Function Arguments

Function Arguments

So far, most named functions have been defined like this:


	function someName () {
	  // ...
	  // ...
	}

...and called like this:


	someName();

Many of you have inquired: What are the () for?

Function Arguments

Arguments: inputs that are "passed" into a function

• Define a function that will accept an argument:


	function hello (name) {
	  alert("Hello, it's " + name);
	}

• Call a function that accepts an argument:


	hello('me');

	// outputs: "hello, it's me"

	hello('Robert');
	hello('Chandler');
Function Arguments

• Function arguments are accessible in the function body using their respective names:


	function hello (name) {
	  alert("Hello, it's " + name);
	}

• Whatever is "passed" as name replaces instances of name in the function body:


	hello('rob');

	// translates to:

	function hello ('rob') {
	  alert("Hello, it's " + 'rob');
	}
Function Arguments: Multiple Arguments

• Functions can accept multiple arguments

• Use a comma-separated list to define:


	function add (x, y) {
	  return x + y;
	}

• Call a function with multiple arguments:


	add(2, 2);
	// returns 4

	add(2, 5);
	// returns 7
Arguments vs Parameters

• Arguments and parameters are often used interchangeably

• There is a clear distinction:

⇒ Parameters refer to the variables declared in the function definition

⇒ Arugments refer to the data passed into the function

Arguments vs Parameters

Parameters:

• Function add defined with two parameters, x and y:


	function add (x, y) {
	  return x + y;
	}

Arguments:

• Function add called with the arguments 5 and 2:


	add(5, 2);

	// returns 7

HTML Insertion

HTML Insertion

.append('content')

⇒ Insert "content" at the END of each element


	$('div').append('<p>Appended paragraph</p>')

.prepend('content')

⇒ Insert "content" at the BEGINNING of each element


	$('div').prepend('<p>Prepended paragraph</p>')
HTML Insertion

• Assuming the following HTML:


	<div id="someDiv">
	  <p>A paragrpah of text</p>
	</div>

• The following JS:


	$('#someDiv').append('<p>Appended paragraph</p>')
	$('#someDiv').prepend('<p>Prepended paragraph</p>')

• Will yield the following HTML:


	<div id="someDiv">
	  <p>Prepended paragraph</p>
	  <p>A paragrpah of text</p>
	  <p>Appened paragraph</p>
	</div>
HTML Insertion

.before('content')

⇒ Insert "content" BEFORE each element


	$('div').before('<p>Before paragraph</p>')

.after('content')

⇒ Insert "content" AFTER each element


	$('div').after('<p>After paragraph</p>')
HTML Insertion

Assuming the following HTML:


	<div id="someDiv">
	  <p>A paragrpah of text</p>
	</div>

The following JS:


	$('#someDiv').before('<p>Before paragraph</p>')
	$('#someDiv').after('<p>After paragraph</p>')

Will yield the following HTML:


	<p>Before paragraph</p>
	<div id="someDiv">
	  <p>A paragrpah of text</p>
	</div>
	<p>After paragraph</p>
