Pseudo Class Selectors

Pseudo Class Selectors

• Pseudo class selectors allow styling given an element's state

• Consider an <a> tag; it can be:

⇒ Unvisited (normal)

⇒ Visited

⇒ Hovered

⇒ Active

• With pseudo classes, we can apply CSS rules to each of the above states

Pseudo Class Selectors

• Pseudo class selectors are applied using the :


	SELECTOR:PSEUDO_CLASS {
	  ...styles
	}

• Pseudo selectors work with id, class, and element selectors:


	a:hover {
	  color: green;
	}

	.nav-link:visited {
	  color: pink;
	}

	#home-link:active {
  	  color: orange;
	}

Floats

First: A History of Positioning

• Since the dawn of the Internet, developers have wanted to be able to move and position elements

• There have been different ways to do this depending on the browsers you need to support:

• Float (IE < 9)

• display: inline-block;(IE < 10)

• Flexbox (IE < 11)

• CSS grid (IE < 11)

• All of these are ways to help arrange elements in a horizonal (or vertical) order.

Floats

• CSS positioning property

• Another tool for rendering block elements inline

• Enables more robust/advanced web layouts

Floats

• Applied as follows:


	.container {
	  float: left;
	}

• Valid values: left, right, none

• Original purpose: allow text to "wrap" around images

• Has become: a technique for advanced web layouts

Clearing Floats

Clearing Floats

• Floated objects do not add height to their containing objects; this creates two problems:

1) Objects further down the page will render incorrectly

2) If there are only floated objects within an object, it will collapse

• To solve these issues, we must handle, or clear, floated elements

• Three methods:

1) overflow: hidden;

2) The clear CSS property

3) The "clearfix"

Handling Floats: The Overflow Method

• Add the following to the collapsed element:

overflow: hidden;

• Tells the browser to pay attention to the height of the content

• Technically isn't clearing...a parent cannot clear its children's floats

• Establishes a new block formatting context

• Elements that create new BFC's are designed to acknowledge their children's floats

Clearing Floats: The Clear Class

1) Define the following in your CSS:


	.clear {
	  clear: both;
	}

2) Create the following element in your HTML; place it AFTER the floated elements


    <div id="floated-right"></div>
    <div id="floated-left"></div>
    <div class="clear"></div>
Clearing Floats: The Clearfix Class

• Define the following in your CSS:


	.clearfix:after {
	  content:"";
	  display:table;
	  clear:both;
	}

• Add the class to the element that has collapsed
